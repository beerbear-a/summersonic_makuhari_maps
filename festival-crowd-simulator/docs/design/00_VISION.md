## 1. タイトル

**正式タイトル（推奨）**: 『メモリーバンド 〜マクハリ、いちにちだけの夏〜』
英題: **MEMORY BAND: One Day Summer at Makuhari**

- 「バンド」= リストバンド と 音楽バンド のダブルミーニング
- ゲームの核となるアイテム「メモリーバンド」（思い出を8つ記録するリストバンド）がそのまま題名になる
- 候補案（没）: 『ぼくのサマーフェス』『MAKUHARI』『フェスの日』

## 2. エレベーターピッチ

> **MOTHER2 × 生きた群衆シミュレーション。**
> 800人の観客がそれぞれの目当てで動き続ける本物のフェス会場で、
> 主人公は「顔も知らないネット友達と、大トリで会う」約束のために一日を過ごす。
> タイムテーブルは待ってくれない。全部は観られない。
> だからこそ、選んだものぜんぶが思い出になる——二度と戻らない夏の一日のRPG。

## 3. 体験ゴール（プレイヤーに何を感じてほしいか）

1. **「時間が惜しい」** — タイテの被り、伸びるトイレ列、腹ペコ。何かを選べば何かを諦める。この小さな損失の連続が、一日を「自分だけの一日」にする
2. **「群衆が生きている」** — NPCはただの背景ではない。800人が本当にライブへ集まり、終演後に流れ、雨で走り出す。その中の一人ひとりに話しかけられる
3. **「笑って、最後にちょっと泣く」** — 昼はバカバカしく、夜は切なく。MOTHER2の感情曲線をなぞる
4. **「音楽が主役」** — ステージに近づくと音が大きくなり、離れると風に混ざる。ラストのフィナーレは音と光の演出でクライマックスを作る

## 4. MOTHER2のDNA → 本作への翻訳表

| MOTHER2の要素 | 本質 | 本作での翻訳 |
| --- | --- | --- |
| どこにでもいる少年ネス | 特別な力のない主人公 | 初めて一人でフェスに来た中学生「ハジメ」（名前変更可）。武器はリストバンドと小遣い5,000円だけ |
| サウンドストーン | 8つのメロディを集める旅の背骨 | **メモリーバンド**: 一日の中の「8つのメモリー」（ライブや出来事の記憶）を記録する。全8つでトゥルーエンド |
| NPCの一言が世界を作る | 会話こそコンテンツ | ペルソナ×時間帯×状況で変化するセリフプール1,000行以上。全員に話しかけたくなる文体（→02章 文体ガイド） |
| ユーモアと切なさの同居 | ふざけた顔で本気の話をする | ケバブ屋の親父のボケの裏に「毎年ここで娘を待っている」等の背景を仕込む |
| 敵もどこか憎めない | 戦闘=世界とのコミカルな衝突 | **戦闘なし**。本作の「敵」は空腹・尿意・炎天下・トイレ大行列・タイテ被り・ゲリラ豪雨。すべて憎めない敵 |
| パパへの電話（セーブ） | セーブに物語と笑いを載せる | **ママからのLINE**でセーブ。「ちゃんと水分とってる？」に既読をつけるとセーブされる |
| どせいさん | 世界の外側にいる癒し | フェス公式マスコット「**ソニ夫くん**」。着ぐるみで会場のヘンな場所に出没。中の人は暑さで限界 |
| カメラマン（チーズ！） | 旅の記録が最後に効いてくる | **シャッター先生**が要所で写真を撮る。エンディングのクレジットで一日分の写真がスライドショーになる |
| 「おとなも こどもも、おねーさんも。」 | 誰にでも開かれた入口 | 死なない・詰まない・急かされすぎない。難易度はルート計画の巧拙にだけ表れる |
| ホームシック | 弱さがシステムになっている | 主人公は疲労が溜まると「家に帰りたく」なる（HYPE減少加速）。ベンチで休む・フェス飯で回復 |

## 5. デザインピラー（迷ったらここに戻る）

1. **一日は一回きり** — 巻き戻しなし。取り逃した記憶は取り逃したまま終わる（周回で回収できる）
2. **群衆シムの上に物語を置く** — 既存の800人エージェント/フローフィールド/混雑システムは物語の「舞台装置」。物語のために群衆を止めたり嘘をつかせたりしない
3. **会話 > カットシーン** — 語りたいことはNPCのセリフに分散させる。ムービーで説明しない
4. **数字より思い出** — スコアは残すが、リザルトの主役は「写真アルバム」と「8つのメモリー」

## 6. アンチゴール（作らないもの）

- ❌ 戦闘・バトルシステム（MOTHER2のガワだけ借りて中身は借りない）
- ❌ シビアなサバイバル（餓死・熱中症死はしない。倒れても救護テントで復帰）
- ❌ 経営シミュレーション化（運営視点は「観察モード」として残すが、主役はプレイヤーの一日）
- ❌ オープンワールド的な引き延ばし（会場は今の1マップで完結。広さより密度）
- ❌ 実在アーティストの肖像・楽曲の再現（名前はタイムテーブル上のテキストのみ。ライブ演出は光と群衆と抽象チップチューンで表現する）
