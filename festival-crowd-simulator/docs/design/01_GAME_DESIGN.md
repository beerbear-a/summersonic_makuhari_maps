# 01. ゲームデザイン — コアループ・システム・バランス

## 1. コアループ

```
        ┌──────────────────────────────────────────────┐
        │  タイムテーブルを見る（次に何を観るか決める）  │
        └──────────────┬───────────────────────────────┘
                       ▼
   ┌─────────── 会場を歩く ────────────┐
   │  ・群衆の流れに巻き込まれる        │
   │  ・NPCに話しかける（世界を知る）   │←─── 欲求（腹・トイレ・水分・疲労）が
   │  ・クエスト/イベントに遭遇する     │      ルートを乱す =「予定通りにいかない」
   └──────────────┬────────────────────┘
                  ▼
        ライブを観る / 出来事に立ち会う
                  ▼
        メモリー獲得・HYPE上昇・写真が残る
                  ▼
        時計は進む。次の被りを選ぶ ──→ ループ先頭へ
```

**1ループの長さ**: 実時間3〜6分（ライブ1本＋移動＋寄り道）。
このループを一日で10〜15回まわし、最後にフィナーレ（スクリプト演出）で締める。

## 2. ゲームモード

| モード | 内容 | 解放 |
| --- | --- | --- |
| **ストーリー** | 本編。朝の自宅プロローグ→フェスの一日→エピローグ | 初期 |
| **スタッフ端末（俯瞰）** | 既存の俯瞰モード。作中では「運営スタッフ用の混雑モニター」という設定でスマホメニューから覗ける。観察専用 | 初期 |
| **フリーローム** | 物語なしで一日を自由に過ごす（現行v1相当）。タイテはシャッフル可 | 1周クリア後 |
| **2日通し券（NG+）** | 周回。所持品と実績を引き継ぎ、タイムテーブルの被りがより意地悪な「2日目」 | トゥルーエンド後 |

## 3. 時間設計

- 内部時刻は現行どおり「0時からの分」。ゲーム内一日は **10:00（自宅）→ 10:30 開場 → 21:20 閉場 → エピローグ**
- **ストーリーモードの時速: 実時間1秒 = ゲーム内0.5分**（現行2.5分/秒から変更。定数 `BASE_MINUTES_PER_SECOND`）
  - 一日（650分）= 素の時計だけなら実時間約22分。会話・カットシーン・メニュー中は**時計停止**、
    ライブ鑑賞中は0.75倍に減速（じっくり観られる）。寄り道込みで1周60〜90分になる
- **時間をスキップする手段**（待ち時間を苦痛にしない）:
  - ベンチ/休憩所で「休む」→ 15分/30分/次のお目当ての15分前 まで送れる（疲労回復ボーナス付き）
  - スキップ中も群衆シムは高速で回す（人の流れが早送りで見える＝ご褒美映像になる）
- 速度ボタン(1x/2x/4x/8x)は**スタッフ端末とフリーロームのみ**。ストーリーでは非表示

## 4. プレイヤーステータス

| ステータス | 範囲 | 上がる要因 | 下がる要因 | 影響 |
| --- | --- | --- | --- | --- |
| おなか(hunger) | 0-100 | 時間経過 | フェス飯 | 80+でHYPE減少、セリフ変化 |
| トイレ(toilet) | 0-100 | 時間経過・飲食 | トイレ | 80+で移動中に「もじもじ歩き」アニメ、HYPE減少 |
| **すいぶん(hydration)** ★新 | 0-100 | ドリンク | 時間経過（晴天時2倍） | 20未満で「くらくら」状態→放置で熱中症イベント |
| つかれ(fatigue) | 0-100 | 移動・鑑賞 | ベンチ・食事 | 80+で歩速-20%、「家に帰りたい」セリフ |
| **HYPE** | 0-100 | ライブ・メモリー・クエスト・買い食い | 混雑ストレス・欲求無視 | エンドランク係数。95+で実績 |

- **熱中症で倒れた場合**: 画面がぐにゃりと歪み、「気がつくと救護テントだった」。ゲーム内30分経過・
  所持ドリンク没収の代わりに看護師NPCの名セリフが聞ける。**死・ゲームオーバーは存在しない**
- MOTHER2の「ころがるHPメーター」オマージュ: 欲求ゲージは即時に減らず**じわじわ追いつく**
  （内部値→表示値をlerp）。ダメージを食らっても走ってトイレに間に合う、が再現できる

## 6. お金

- 開始時 **¥5,000**（ママからのお小遣い）
- 使い道: 飲食・グッズ・便利アイテム。**グッズ（¥3,500のTシャツ等）を買うと便利アイテムが買えなくなる**
  ジレンマが夏の小遣いのリアル
- 収入: サイドクエスト報酬（お礼に500円くれる大人、など）
- **ママPayPay**: 一日一回だけ、LINEで「お金足りてる？」と聞かれ「たりない…」を選ぶと¥2,000送金+
  ちょっと切ないセリフ。MOTHER2のパパATMオマージュ

## 7. メモリーシステム（サウンドストーン相当・本作の背骨）

リストバンド型デバイス「メモリーバンド」に、一日の中で**8つのメモリー**を記録する。
メモリー獲得時は専用ジングル＋画面フラッシュ＋バンドのLEDが1つ点灯する演出。

| # | メモリー名 | 獲得条件 | 獲得可能時間帯 |
| --- | --- | --- | --- |
| 1 | 朝いちばんの大歓声 | Mrs. GREEN APPLE(MARINE 11:00)を最前で観る | 11:00-11:45 |
| 2 | 路上のマクハリズム | 駅前で路上ライブ中のインディーバンドのクエスト第1話を見る | 10:30-12:30 |
| 3 | 海風のビート | BEACH STAGEでいずれかのアクトを観る | 12:00-18:10 |
| 4 | 雨あがりの虹 | ゲリラ豪雨イベントを最後まで体験する（ポンチョの有無は不問） | 発生時(±15:00) |
| 5 | 屋台のいちばん星 | フェス飯を3種類食べ、ケバブ屋の親父のクエストを完了する | 終日 |
| 6 | 夕暮れのシンガロング | 優里(MOUNTAIN 19:00)で観客の大合唱に立ち会う | 19:00-19:50 |
| 7 | まいごの音符 | 迷子のリコを家族に届けるクエストを完了する | 13:00-17:00 |
| 8 | 花火とフィナーレ | ALICIA KEYS(MARINE 19:30-20:50)のフィナーレをスタジアムで観る | 19:30-20:50 |

- **設計上の要点**: 6と8は時間が被る（19:00-19:50 vs 19:30-20:50）。優里を途中で抜けて
  スタジアムへ走れば両取り可能——**ここが一日の最大のルート判断**。ロックおじさんがヒントをくれる
- 8つ全て+ミライとの合流でトゥルーエンド。7つ以下は通常エンド（それでも温かく終わる）
- メモリーはスマホメニューの「バンド」画面でいつでも見返せる（獲得時の写真つき）

## 8. クエスト

- **メインクエスト1本**（ミライと大トリで会う约束）+ **サイドクエスト15本**（→02章に全リスト）
- クエストUI: スマホの「やること」メモに自動追記。**矢印やマーカーで誘導しない**
  （NPCのセリフと地名で探させる。MOTHER2の「ひとに聞く」感覚。ただしメモには要点が残る）
- 報酬はお金・アイテム・メモリー・セリフ変化。**経験値やレベルは存在しない**

## 9. ダイナミックイベント（EventDirector）

一日に彩りを与えるスクリプト/ランダムイベント。同時発生は最大1つ。

| イベント | 発生条件 | 内容 |
| --- | --- | --- |
| **ゲリラ豪雨** | 14:30-15:30の間に70%で発生（未発生なら16:00に確定発生） | 10分前に空が暗転、売り子がポンチョ販売開始。降雨中は屋外の全員がメッセ/屋根へ殺到（群衆シムの見せ場）。15分後に止み、**虹**が出て「メモリー4」チャンス |
| アンコール | 人気80+のアクト終演時 30% | 終演が10分延長。次のアクトを狙う移動計画が狂う |
| シークレットゲスト | 16:30 PACIFICの空き枠で確定 | タイテに「?」が出現、当日発表の風。観客がざわつく |
| 迷子放送 | リコのクエスト進行に連動 | 会場アナウンス（画面上部テロップ）が流れる |
| ウェーブ | MARINEでライブ中、観客密度が閾値超え | スタジアムの観客が順にジャンプするウェーブ演出。参加するとHYPE+ |
| グッズ完売タイマー | 13:00にTシャツ残数表示開始 | 17:00で完売。買うか小遣いを残すかの駆け引き |
| 熱中症注意報 | 12:30固定 | アナウンス+NPC全体のセリフが水分系に変化。ドリンク売店に列ができる |

## 10. エンディングとランク

エンディングは「どう過ごしたか」で分岐。ランク(S〜D)は継続するが主役ではない。

| エンド | 条件 | 内容 |
| --- | --- | --- |
| **トゥルー「いちにちだけの夏」** | メモリー8/8 + フィナーレでミライと合流 | 花火のあと、ミライの正体が明かされる（→02章）。写真スライドショー→エンドロール→「また来年。」 |
| **通常「かえりの電車」** | フィナーレを観た | 満員電車で寝落ちする主人公。獲得したメモリーの数だけ夢に断片が流れる |
| **途中退場「まだ帰りたくない」** | 閉場前に駅から帰るを選択 | 途中で帰った人向けの静かな終わり。実績「また来ればいいさ」 |
| **裏「クローズドサーキット」** | 閉場後、退場せず23:00まで隠れて残る（NG+のみ） | 誰もいない会場でソニ夫くん(中の人)と語る隠しシーン |

## 13. 難易度・アクセシビリティ

- 詰み・死・時間切れによる強制BADなし。全部を取れないだけで、何をしても「一日」は完成する
- 漢字にはふりがな不要の語彙を優先（小学生高学年が読める文）
- 色弱対応: 状態色に依存する情報は必ずセリフ/アイコンでも表現
- ボタンは決定(Z/Space/Enter)・キャンセル(X/Esc)・メニュー(C/Tab)の3系統に集約
