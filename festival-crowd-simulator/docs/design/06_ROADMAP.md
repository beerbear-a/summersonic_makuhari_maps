# 06. 実装ロードマップ — マイルストーン・受け入れ条件・リスク

## 0. 進め方の原則

- **各マイルストーンの終わりに必ず「遊べる状態」を維持する**（縦切り。横に作らない）
- 1マイルストーン = 1〜数コミットのまとまり。完了ごとに05章7節のPlaywright検証を回す
- 迷ったら 00章のピラーへ。特に「会話 > カットシーン」「数字より思い出」

## M1. 基盤（GameState / セーブ / インベントリ / お金）

**スコープ**
- main.ts の分割（ui/Hud, ui/DialogueWindow抽出、GameState導入、時計停止機構）
- ストーリー時速への変更（0.5分/秒・会話中停止・ベンチ時間送り）
- Inventory(8スロット)+items.ts、所持金、売店での購入フロー（会話UIに「かう」選択肢）
- hydrationステータス+熱中症→救護テント遷移
- SaveSystem+ママLINEセーブ（LINEタブUIは仮でよい）
- SCOREのHUD撤去（内部計算は維持）

**受け入れ条件**
- 買う→食べる→ゲージ回復→セーブ→リロード復元 が通しで動く
- 熱中症で倒れても30分後に復帰しゲーム続行できる
- `npm run build` グリーン、通し検証1・6パス

## M2. 会話・ペルソナ・クエスト

**スコープ**
- DialogueEngine（ツリー/条件/選択肢/フラグ）、顔チップ、感情バブル
- PersonaController+12人のスケジュール移動と専用配色スプライト
- QuestSystem+サイドクエスト15本のうち**まず5本**（#1,2,4,5,6: チュートリアル/護送/バンド系を優先）
- スマホメニュー（マップ/タイテ/やること/もちもの/LINEの5タブ）
- モブセリフプールの時間帯4×状態拡張（300行目標のうち150行）

**受け入れ条件**
- 「まいごのリコ」を通しでプレイでき、メモリー7が点灯する
- セトリ姉妹が実際に別ステージ間を移動している（俯瞰で確認）
- 通し検証2パス

## M3. イベント・天候・音楽

**スコープ**
- EventDirector+ゲリラ豪雨（WeatherLayer: 雨/虹）、アンコール、シークレットゲスト、
  グッズ完売、熱中症注意報、ウェーブ
- ChipTracker+ステージ距離クロスフェード+アンビエント昼夜（全曲でなくてよい、
  まずMARINE/MOUNTAIN/アンビエント2種/豪雨の5曲）
- 群衆介入API3種（overrideWeather / extendAct / injectAct）

**受け入れ条件**
- 豪雨で群衆が屋根に殺到→虹→メモリー4獲得の一連が動く（seed固定テスト）
- ステージへ歩いて近づくと音楽がクロスフェードする
- 通し検証3パス

## M4. 物語（プロローグ〜フィナーレ〜エンディング）

**スコープ**
- タイトル画面・名前入力・プロローグ（自宅）・エピローグ
- ミライの5遭遇+メインクエスト、メモリー8種の獲得ロジック完成
- CutsceneDirector+カットシーン7本、フィナーレ演出（ペンライト・静寂・花火）
- エンディング4種分岐、リザルト改修（写真スライドショー）
- 残りのサイドクエスト10本、セリフ1,000行完成、実績24種

**受け入れ条件**
- 通し検証4・5パス（**優里→アリシア両取りが本当に間に合う**ことを最重要確認）
- 初見想定プレイでトゥルー到達に必要な情報がNPCセリフから得られる（ロックおじさん等）

## M5. 磨き（ゲームフィール・バランス・NG+）

**スコープ**
- 03章10節のフィードバック全部盛り、設定画面、文字送り速度
- 全曲実装（フィナーレ組曲・ミライのテーマ織り込み）
- NG+（2日通し券）、フリーローム、裏エンド
- バランス調整（01章12節の初期値からプレイテストで詰める）、モバイル動作確認（任意）

**受け入れ条件**
- 1周60〜90分に収まる／全実績が取得可能／全エンド到達可能
- 60fps（開発機実測）を大きく割るシーンがない

## リスク台帳

| リスク | 兆候 | 対策 |
| --- | --- | --- |
| セリフ量産の質劣化 | 文体ガイド違反の説明口調 | 02章6節の見本をレビュー基準にする。1画面3行厳守 |
| 豪雨の群衆殺到でフレーム落ち | 全員が同時に再経路探索 | 屋根フィールドは事前計算1本を共有。再decide()は毎フレーム上限50体に分散 |
| 両取りバランス崩壊 | 検証4が不安定 | プレイヤー歩速/優里終演/アリシア開演のどれか1つだけで調整（複数同時に触らない） |
| スコープ肥大 | M4が2倍に膨らむ | サイドクエストは15→10本に減らしてよい（#7,8,13,14,9が切り捨て候補）。カットシーン7本は死守 |
| セーブ互換 | スキーマ変更で壊れる | SaveData.versionで移行、読めない場合は「思い出は消えない」ダイアログ→ニューゲーム |
