## 実装担当への最初の3ステップ

1. 00章と05章1節（不変条件）を読む — 特に「座標は必ずWORLD_SCALE込み」
2. 06章のM1から着手する。**縦切り**（常に遊べる状態を保つ）
3. セリフを書くときは02章6節の文体ガイドを開いたままにする
